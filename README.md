Atividade para integração do github no grupo.
As funções devem ser implementadas em bibliotecaGAAP.py e cada um deve criar um teste para a própria função em main.py.

**Tarefas:**

Ideia do projeto: Fazer em um único arquivo uma biblioteca em python com funções relacionadas a química.

IMPORTANTE: Cada membro deve criar uma branch própria baseada na main, para que após terminada a implementação da função poder fazer o
próprio PullRequest, mas não confirmar o merge, deixar merge para o responsável.
